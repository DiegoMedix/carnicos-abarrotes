# 🥩 Cárnicos & Abarrotes Don Luis

Sitio web desarrollado como proyecto práctico del taller de GitHub.

## 🌐 Sitio publicado

https://diegomedix.github.io/carnicos-abarrotes/

## 📁 Estructura del proyecto
carnicos-abarrotes/
├── index.html        → Página principal
├── productos.html    → Catálogo de productos
├── contacto.html     → Información y formulario de contacto
├── css/
│   └── estilos.css   → Estilos del sitio
└── README.md         → Documentación del proyecto

## 🛠️ Tecnologías usadas

- HTML5
- CSS3
- GitHub Pages (deploy)

## 🔀 Flujo de trabajo Git utilizado

1. Creación del repositorio en GitHub
2. Desarrollo de archivos en la rama `feature/catalogo`
3. Pull Request de `feature/catalogo` → `main`
4. Merge y publicación automática con GitHub Pages
